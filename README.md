# Android UI Showcase

A premium portfolio project demonstrating high-end, modern Android UI design using **Jetpack Compose** and **Material 3**. This project features three distinct industry-focused screens, each with its own unique design language, color palette, and component architecture.

## 📱 The Showcase

### 1. Tastique (Food Delivery)
*   **Vibe:** Dark, luxury, high-contrast.
*   **Focus:** Rich food photography, moody aesthetics, and amber accents.
*   **Key Components:** Hero dish headers, category filtering, menu item cards with ratings, and a stylized shopping cart experience.

### 2. MediCare (Health & Wellness)
*   **Vibe:** Clean, trustworthy, professional.
*   **Focus:** Clarity, accessibility, and soothing teal tones.
*   **Key Components:** Doctor profiles, appointment scheduling (date/time selectors), specialty categories, and search functionality.

### 3. Drape (Fashion Retail)
*   **Vibe:** Bold, minimalist, editorial.
*   **Focus:** High-fashion product presentation and vibrant coral accents.
*   **Key Components:** Product grids, favorite toggles, search bars, and clean typography.

### 4. The Hub
*   A central dashboard that serves as a launchpad for the different UI demos, featuring tinted cards and a unified navigation structure.

---

## 🛠 Tech Stack & Architecture

*   **100% Jetpack Compose:** Declarative UI built entirely in Kotlin.
*   **Material 3:** Leveraging the latest Material design components and dynamic color principles.
*   **Advanced Theming:** Each showcase screen implements its own **private MaterialTheme override**. This allows for multiple distinct design systems to coexist within a single app.
*   **Navigation Compose:** Type-safe navigation (Navigation 2.8+) for seamless transitions between screens.
*   **Coil:** Modern, asynchronous image loading for high-quality remote assets.
*   **Clean Organization:** Package-by-feature structure with self-contained screen implementations, showcasing how to build modular, maintainable UI code.

---

## 🚀 Getting Started

1.  Clone the repository.
2.  Open in **Android Studio Ladybug** or newer.
3.  Sync Gradle and run on an emulator or physical device.
